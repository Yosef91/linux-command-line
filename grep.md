grep is a command that helps you search for stuff. take ls. if you do ls, it wil print out files in you directory. here is an example of ls using grep:

ls|grep pizza

this will print out stuff only that include the word "pizza" in it.

grep can be used for history to. for example:

history|grep hamburger

it will only print out past commands that include hamburger.
